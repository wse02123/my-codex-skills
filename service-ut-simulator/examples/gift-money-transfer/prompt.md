# 스킬 실행용 복사 텍스트

아래 내용을 `service-ut-simulator` 스킬 실행 후 입력해보세요.

```text
1. 화면 프로세스 이미지 또는 목업
- 첨부 이미지: screen-flow.svg

2. 화면 순서
- 이체 완료 화면 → 경조금 카테고리 선택 바텀시트 → 경조사메모지 등록 완료 상태

3. 테스트 사용자 맥락
- 기존 사용자

4. 검증할 기능과 사용자가 완료해야 하는 핵심 행동
- 이체를 한 사용자가 경조금을 이체했다면, 경조금 카테고리를 지정하여 경조사메모지에 등록할까?

5. 현재 예상 타겟
- 30대 결혼식을 자주 참가하는 사람
- 50대 장례식을 자주 참가하는 사람
```

선택정보까지 함께 넣고 싶다면 `README.md`의 선택정보 예시를 같이 전달하세요.
